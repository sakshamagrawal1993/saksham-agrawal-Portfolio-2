# Independent review assignment

Review Health Twin changes against `main`, the PRD, the acceptance contract, and
the current Codex plan.

Focus on:

- User-visible correctness and regressions
- Cross-user data isolation and storage ownership
- Edge-function and n8n contracts
- Secret handling
- Missing negative/error-path coverage
- False-positive or superficial tests
- Playground mutations of real data
- Chat context leakage
- Incomplete controls presented as working functionality

Do not modify files. Findings must be specific, actionable, and tied to files,
lines, or acceptance IDs. Do not treat unavailable live evidence as passing.
